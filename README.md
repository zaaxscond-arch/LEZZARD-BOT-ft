

# 🗿 LEZZARD BOT

WhatsApp Bot with **Kudeta** (Auto Report Group) + **Sticker** features.

## 🔥 Features
- `/kudeta [link]` — Auto report group members (tangguhkan grup)
- `/sticker [url]` — Create sticker from image (reply or URL)
- `ping` — Check bot connection
- `help` — Show all commands

## 🚀 Installation

```bash
# Clone repository
git clone https://github.com/usernamekamu/lezzard-bot.git
cd lezzard-bot

# Install dependencies
npm install

# Pair WhatsApp
npm run pair 6282229038075

# Start bot
npm start
